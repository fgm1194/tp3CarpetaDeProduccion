Ejercicio 06

1. Cambiar el color de fondo para pantallas grandes:
Objetivo: Cambiar el color de fondo de la página a un color más oscuro para pantallas con un ancho mínimo de 960px.
Media Query: @media (min-width: 960px)
Pista: Usa una clase CSS para definir el color de fondo y aplícala dentro del Media Query.
2. Ajustar el tamaño de la fuente para pantallas pequeñas:
Objetivo: Aumentar el tamaño de la fuente del texto principal para pantallas con un ancho máximo de 600px.
Media Query: @media (max-width: 600px)
Pista: Define un tamaño de fuente más grande en una clase CSS y aplícala dentro del Media Query.
3. Esconder elementos en orientación vertical:
Objetivo: Ocultar un elemento específico (por ejemplo, una barra lateral) cuando la pantalla esté en orientación vertical (modo retrato).
Media Query: @media (orientation: portrait)
Pista: Usa la propiedad display: none para ocultar el elemento dentro del Media Query.
4. Mostrar elementos en orientación horizontal:
Objetivo: Mostrar un elemento específico (por ejemplo, una barra de navegación) solo cuando la pantalla esté en orientación horizontal (modo paisaje).
Media Query: @media (orientation: landscape)
Pista: Usa la propiedad display: block para mostrar el elemento dentro del Media Query.
5. Cambiar el diseño de columnas con Flexbox para pantallas pequeñas:
Objetivo: Cambiar la disposición de elementos en columnas a una disposición en una sola columna para pantallas con un ancho máximo de 768px.
Media Query: @media (max-width: 768px)
Pista: Usa las propiedades de Flexbox flex-direction: column y align-items: center para la nueva disposición.
6. Aumentar el espacio entre elementos en pantallas grandes:
Objetivo: Aumentar el margen entre elementos de una lista o sección para pantallas con un ancho mínimo de 1200px.
Media Query: @media (min-width: 1200px)
Pista: Define valores más grandes para las propiedades de margen dentro del Media Query.
7. Cambiar la tipografía para títulos en pantallas pequeñas:
Objetivo: Cambiar la familia de fuentes para títulos a una fuente más legible en pantallas con un ancho máximo de 576px.
Media Query: @media (max-width: 576px)
Pista: Define una nueva familia de fuentes para los títulos dentro del Media Query.
8. Ajustar el tamaño de imágenes para pantallas pequeñas:
Objetivo: Reducir el tamaño de las imágenes para que no ocupen todo el ancho de la pantalla en dispositivos móviles.
Media Query: @media (max-width: 480px)
Pista: Usa la propiedad width con un porcentaje o valor máximo dentro del Media Query.
Consejos adicionales:
Combina diferentes Media Queries para crear diseños adaptativos para distintos rangos de tamaños de pantalla.
Utiliza herramientas de desarrollo web para previsualizar tu sitio web en diferentes dispositivos y ajustar tus Media Queries en consecuencia.
Investiga las últimas propiedades y técnicas de CSS para crear diseños adaptables modernos.
