Ejercicio 10

En esta actividad, implementarán cinco plugins de jQuery en los archivos de ejercicios previos de Position con CSS. Los plugins añadirán una característica o componente interactivo en cada archivo.

Requisitos Generales
  • Selección de Plugins: Seleccionen e implementen un plugin para cada tipo de funcionalidad especificada a continuación. Mencionen únicamente qué plugin eligieron en cada caso.
  • Implementación: Apliquen cada plugin de forma básica o, si el plugin lo permite, personalícenlo usando las opciones configurables que ofrece.
  • Uso de Bibliotecas
    Adicionales    : Se permite el uso de bibliotecas adicionales solo si el plugin las requiere, pero esto no es obligatorio.
Ejercicios

Carrusel de Imágenes:
  • Seleccionen un plugin de carrusel en jQuery y aplíquenlo en el ejercicio de Position.
  • Integren imágenes y configuren opciones básicas de navegación.
Integración con WhatsApp:
  • Seleccionen un plugin que permita enlazar WhatsApp.
  • Inserten un botón de contacto que redirija al usuario a WhatsApp con un mensaje predefinido al hacer clic.
Animación Parallax de Fondo:
  • Seleccionen un plugin que permita añadir un efecto parallax en el fondo.
  • Configuren el fondo de una sección para que tenga una animación parallax básica.
Lightbox:
  • Seleccionen un plugin que permita crear un lightbox para mostrar imágenes en pantalla completa.
  • Apliquen el plugin a una imagen o galería de imágenes en el archivo de ejercicio.
Slider de Contenido:
  • Seleccionen un plugin de slider que permita desplazar contenido horizontal o verticalmente.
  • Configuren el slider para mostrar texto o imágenes en secuencia.
