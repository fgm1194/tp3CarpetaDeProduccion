Ejercicio 05

A partir de los documentos HTML que se proporcionan, debes escribir las reglas CSS necesarias para lograr 2 maquetados web que tengan el mismo aspecto (o lo más cercano posible) que la imagen adjunta.

El maquetado debe realizarse 2 veces, con técnicas diferentes: Fijas y Fluidas. El CSS se escribe en cada HTML mediante un bloque <style>

Maquetado Fijo
Las características del diseño de esta página son:
  • Un diseño fijo centrado con un ancho de 960 px.
  • El encabezado de nivel 1 centrado.
  • El contenido principal de la página se organiza en dos columnas, una de 260 px de ancho y la otra el espacio restante de 700 px.
  • La lista que explica el contenido del libro se muestra como un texto a dos columnas del mismo ancho.
Maquetado Fluido
Las características del diseño de esta página son:
  • Un diseño líquido centrado con un ancho del 80%.
  • El encabezado de nivel 1 centrado.
  • El contenido principal de la página se organiza en dos columnas, una de 25% de ancho y la otra el espacio restante de 75%.
  • La lista que explica el contenido del libro se muestra como un texto a dos columnas del mismo ancho.
En cada caso, se puede modificar el HTML, para agregar estructuras <div>, selectores de clases o ID

DESAFÍO: Para quienes ya conocen sobre propiedades flexbox o grid, en este ejercicio no pueden utilizarse
