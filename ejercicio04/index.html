Ejercicio 04

Utilizando el material adjunto, agregar en el subdirectorio CSS un archivo de estilos donde se escriban todas las reglas utilizadas para trabajar los enunciados descriptos en los textos de cada sección. En el subdirectorio IMG se podrán guardar todas las imágenes a utilizar, a libre elección. El HTML sólo se puede editar, si consideras que necesitas agregar atributos ID o CLASS las las etiquetas ya provistas (no se pueden agregar otras etiquetas, tampoco modificar los estilos predeterminados). Cada <section> debe presentar su propia resolución del ejercicio enunciado.
